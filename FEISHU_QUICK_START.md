# 培训内容同步到飞书 - 快速开始

## 🎯 最简单的方案（推荐）

### 方案：直接在飞书中分享培训网站链接

**优点：**
- ✅ 无需配置，立即可用
- ✅ 内容实时更新，无需同步
- ✅ 保留所有交互功能（图表、图片、链接）
- ✅ 支持手机/平板/电脑访问

**操作步骤：**

1. **在飞书群聊中发送培训链接**
   ```
   🎓 液冷连接系统专家顾问特训营
   
   在线培训网站已上线！
   
   🔗 网站地址：
   https://lucylook6-design.github.io/liquid-cooling-training/
   
   📚 培训内容：
   • Day 1：液冷基础与行业生态（4个模块）
   • Day 2：产品深度解析（4个模块）
   • Day 3：客户场景与解决方案（4个模块）
   • Day 4：顾问式销售技巧（4个模块）
   • Day 5：实战演练与考核（4个模块）
   
   ⏱️ 总时长：5天（40小时）
   🎯 目标：达到行业Top 20%的专家级顾问型销售能力
   
   请大家收藏链接，开始学习！
   ```

2. **在飞书云文档中创建快捷方式**
   - 打开飞书云文档
   - 创建文件夹"培训资料"
   - 点击"新建" → "快捷方式"
   - 粘贴培训网站链接
   - 命名为"液冷培训网站"

3. **添加到飞书工作台**
   - 打开飞书工作台
   - 点击"添加应用"
   - 选择"网页应用"
   - 输入培训网站链接
   - 团队成员可以直接从工作台访问

## 🔄 如果确实需要同步到飞书云文档

### 方案A：使用飞书MCP（自动化）

**前提条件：**
- 需要创建飞书应用（App ID + App Secret）
- 需要配置MCP插件
- 适合技术团队

**详细步骤：** 参考 `FEISHU_SYNC_GUIDE.md`

### 方案B：手动复制粘贴（简单快速）

**步骤：**

1. **创建文件夹结构**
   ```
   液冷培训/
   ├── Day 1 - 液冷基础与行业生态/
   ├── Day 2 - 产品深度解析/
   ├── Day 3 - 客户场景与解决方案/
   ├── Day 4 - 顾问式销售技巧/
   └── Day 5 - 实战演练与考核/
   ```

2. **复制内容**
   - 打开培训网站的某个模块
   - 全选内容（Cmd+A / Ctrl+A）
   - 复制（Cmd+C / Ctrl+C）
   - 在飞书文档中粘贴（Cmd+V / Ctrl+V）

3. **调整格式**
   - 飞书会自动识别标题、列表、表格
   - 手动调整图片和链接

**注意：** 这种方式需要手动操作20次（每个模块一次）

## ❓ 常见问题

### Q1：培训网站会更新吗？
**A：** 会的。我们会定期更新：
- 最新的GPU数据
- 新的客户案例
- 优化的销售话术

更新后，链接不变，内容自动更新。

### Q2：可以离线学习吗？
**A：** 可以。在浏览器中：
1. 打开培训网站
2. 按 Cmd+S / Ctrl+S 保存网页
3. 选择"网页，全部"
4. 离线时打开保存的文件

### Q3：可以打印吗？
**A：** 可以。在浏览器中：
1. 打开某个模块
2. 按 Cmd+P / Ctrl+P
3. 选择"另存为PDF"
4. 保存到本地或上传到飞书

### Q4：手机上看不清表格怎么办？
**A：** 培训网站支持响应式设计：
- 横屏查看表格更清晰
- 或者在电脑上学习
- 或者下载PDF版本
