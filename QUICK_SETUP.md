## 📝 第1步：创建飞书应用（2分钟）

### 1.1 访问飞书开放平台

在浏览器中打开：
```
https://open.feishu.cn/
```

### 1.2 创建应用

1. 点击【控制台】
2. 点击【创建企业自建应用】
3. 填写：
   - 应用名称：`液冷培训同步工具`
   - 应用描述：`用于同步培训内容到飞书云文档`
4. 点击【创建】

### 1.3 获取凭证

在应用详情页：
1. 找到【凭证与基础信息】
2. 复制保存：
   - **App ID**（形如：cli_xxxxxx）
   - **App Secret**（点击查看并复制）

### 1.4 添加权限

1. 点击【权限管理】
2. 搜索并勾选：
   - `docx:document`
   - `drive:drive`
3. 点击【发布版本】

### 1.5 配置重定向URL

1. 点击【安全设置】
2. 在【重定向URL】中添加：
   ```
   http://localhost:3000/callback
   ```
3. 点击【保存】

✅ **完成！** 您已获得App ID和App Secret

## 🔧 遇到问题？

### 问题：登录失败

**检查：**
- App ID和App Secret是否正确
- 应用权限是否已发布
- 重定向URL是否为 `http://localhost:3000/callback`

**解决：**
重新运行 `./setup-feishu-mcp.sh`

### 问题：无法创建文档

**检查：**
- 应用权限是否包含 `docx:document` 和 `drive:drive`
- 权限是否已点击【发布版本】

**解决：**
在飞书开放平台重新发布权限，然后重新登录：
```bash
npx -y @larksuiteoapi/lark-mcp login -a <your_app_id> -s <your_app_secret> --scope "offline_access docx:document drive:drive"
```

### 问题：Kiro中看不到飞书MCP工具

**检查：**
- 是否已重启Kiro
- `~/.kiro/settings/mcp.json` 配置是否正确

**解决：**
1. 检查配置文件
2. 重启Kiro
3. 在Kiro中说"列出所有可用的MCP工具"
